# Wine Quality Prediction Using Machine Learning

## Project Overview
This project predicts wine quality using Machine Learning algorithms on the UCI Wine Quality Dataset.

The project includes:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Outlier analysis
- Feature scaling
- Model training
- Cross-validation
- Classification metrics
- Ensemble learning

---

## Datasets Used

1. Red Wine Dataset
2. White Wine Dataset

Both datasets were combined for analysis and prediction.

---

## Machine Learning Models Used

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

---

## Best Performing Model

Random Forest Classifier achieved the best overall performance.

---

## Techniques Implemented

- Missing value analysis
- Duplicate removal
- Label encoding
- Standardization using StandardScaler
- Train-test split
- Cross-validation
- Hyperparameter tuning
- Confusion matrix analysis
- Feature importance analysis
